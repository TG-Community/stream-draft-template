# Contributing

This repository relates to activities in The Gathering Stream.

All material in this repository is considered Contributions to The Gathering
standards process and is subject to The Gathering legal and policy terms:
https://tg.community/privacy-and-policies

Any edit, commit, pull request, issue, comment, or other change made to this
repository is considered a contribution under those terms.

## Workflow

1. Rename `draft-todo-author-stream.md` to your draft filename (it must start
   with `draft-`).
2. Edit the draft source markdown.
3. Use pull requests for review if you are collaborating.
4. Create a GitHub release with a tag ending in a two-digit draft version,
   such as `draft-example-topic-00`, to publish a version.
5. Wait for GitHub Actions to publish the editor's copy and sync the release
   to Stream.
